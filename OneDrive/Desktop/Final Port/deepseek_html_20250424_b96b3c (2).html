Lirante - Food Delivery Solution

UI Design Mobile App Food Delivery

Lirante is a comprehensive food delivery solution designed to connect restaurants with customers through an intuitive mobile interface. The project involved:

  • User research and persona development
  • Wireframing and prototyping
  • UI design with a focus on food imagery
  • Checkout process optimization
  • Delivery tracking system

The app achieved a 4.9/5 average rating on app stores and increased order volume by 35% for partner restaurants.

JX Design System

Design System UI Components Figma

The JX Design System was created to provide a consistent design language across all company products. Key features include:

  • Comprehensive color palette with accessibility checks
  • Typography scale for multiple device sizes
  • Reusable component library
  • Design tokens for development handoff
  • Documentation for designers and developers

The system reduced design time by 40% and improved consistency across all digital products.

Cinetrade Platform

Web App Dashboard Finance

Cinetrade is an innovative platform for investing in digital media assets. The project involved:

  • Complex data visualization for investment tracking
  • Secure authentication flows
  • Portfolio management interface
  • Real-time market data display
  • Responsive design for desktop and mobile

The platform successfully onboarded over 10,000 users in its first 3 months with a 95% satisfaction rate.

Sugee Loan Management

Web Design Finance Rural Sector

Sugee is a loan management system designed specifically for the rural sector, featuring:

  • Simplified application process for non-tech users
  • Multilingual support
  • Offline functionality for areas with poor connectivity
  • Visual loan repayment tracking
  • Accessibility features for low-literacy users

The system has helped disburse over $5M in loans to rural entrepreneurs with a 98% repayment rate.
